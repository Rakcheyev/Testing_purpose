# PBIP Artifacts

У цій папці зберігаються PBIP-файли, M-код, DAX-артефакти, які міроряться з Fabric/Power BI для рев'ю, тестування та стандартизації.

## Призначення
- Локальне рев'ю та перевірка відповідності стандартам (lint, MCP validation)
- Тестування модулів автоматичної стандартизації
- Зручне ведення історії змін через git
- Підготовка до деплою у Fabric/Power BI після проходження рев'ю

## Як використовувати
1. Експортуйте артефакти з Fabric/Power BI у цю папку.
2. Створюйте окремі гілки для рев'ю та тестування.
3. Використовуйте MCP-модулі для автоматичної перевірки.
4. Після схвалення — деплойте у Fabric/Power BI.

## Структура
- `pbip/` — PBIP-файли проектів
- `m_query/` — M-код (Power Query)
- `dax/` — DAX-артефакти (міри, шаблони)
- `reports/` — звіти рев'ю та стандартизації
